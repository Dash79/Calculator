# JavaFX Calculator

JavaFX Calculator is a simple yet functional calculator application built using Java and JavaFX. The application provides a graphical user interface (GUI) that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Key Features

- **Basic Arithmetic Operations:** Perform addition, subtraction, multiplication, and division.
- **Responsive GUI:** Built with JavaFX for a clean and interactive user experience.
- **Clear Display:** Shows input and results clearly on a digital screen interface.
- **Error Handling:** Handles invalid inputs and division by zero with appropriate error messages.

## Prerequisites

Before you can run the application, make sure you have the following installed:

- **Java Development Kit (JDK):** Version 8 or later
- **JavaFX:** Included with the JDK or as a separate module depending on your setup
